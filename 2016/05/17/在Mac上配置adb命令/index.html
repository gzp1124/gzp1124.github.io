在Mac上配置adb命令

发表于 2016-05-17   |   分类于 小知识点   |  

首次在命令行使用adb命令，会提示adb命令找不到
这个时候就要进行配置adb

配置步骤如下：

  • 打开命令行创建 .bash_profile文件（可以先执行第二步，如果 .bash_profile文件不存在再执行第一步）
  • 编辑 .bash_profile文件

  • 在弹出的页面输入：

export PATH=${PATH}:~/Library/Android/sdk/platform-tools

如图：

  • command S保存页面，再打开命令行，adb已经可以使用了。
